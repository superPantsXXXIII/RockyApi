Api - Documentation/Routes

method route info
get thisUrl/rocks?page=1 get the entire collection divided into pages of 10
get thisUrl/rocks/single/:id get a single item by item id
get thisUrl/rocks/search?s=SEARCHTERM get the items that includes in their name or info the SEARCHTERM that was inputed
get thisUrl/rocks/category/:catName get all items in a specified category
get /rocks/priceRange?min=NUM&max=NUM get all items where their price lay bettwen the minimum and maximum inputed
post thisUrl/rocks post an item as a registered user(send auth token through header as x-api-key).
when sending the Json through the body the following fields are Required(name,info,category,price),
and its optional to add an image with the field img_url
put thisUrl/rocks/:itemId put(update) any item you posted as a registered user(send auth token through header as x-api-key, same fields applay as the in the post method)
delete thisUrl/rocks/:itemId delete any item you posted as a registered user
register and support the site
get thisUrl/users/userInfo see your information as it was registered in the site
post thisUrl/users register to our site by sending through the request's body a Json that includes your name, email and password
post thisUrl/users/login login to get your auth token that is required for executing some of the methods in the site by sending through the request's body a Json that includes your email and password
